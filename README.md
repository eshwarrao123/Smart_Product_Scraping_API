# Smart Product Scraping API

A scalable FastAPI-based backend system for extracting, caching, and serving product data using web scraping techniques.

## Features
- RESTful APIs built with FastAPI
- Asynchronous web scraping
- Redis caching for performance optimization
- MongoDB for persistent storage
- Structured data validation using Pydantic

## Tech Stack
Python, FastAPI, Redis, MongoDB

## Future Improvements
- JWT Authentication
- Advanced filtering & search
- Deployment on AWS
